# Ancient World RPG

一款以「大同與小康」為背景的 RPG 遊戲。

---

## 遊戲介紹

玩家將穿越到古代世界，探索村莊、森林與戰場。

途中會遇到：

- 黃帝
- 蚩尤
- 村民
- 神秘怪物

---

## 遊戲特色

### 白天黑夜系統
- 晚上村民會回家
- 城門會關閉
- 地圖霧氣會變濃

### 戰鬥系統
- Boss 二階段變身
- 子彈攻擊
- Undertale 風格閃避

### 任務系統
- 收集豬肉與稻米
- 解鎖特殊劇情
- 隱藏支線

---

## 操作方式

| 按鍵 | 功能 |
|---|---|
| WASD | 移動 |
| Shift | 衝刺 |
| Enter | 對話 |
| ESC | 開啟選單 |
